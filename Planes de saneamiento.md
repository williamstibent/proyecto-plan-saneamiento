# Planes de saneamiento

## Notas rápidas
Exclusivo para cada empresa por que los alimientos son distintos, bajo o alto riesgo.

4 etapas, o planes que tienen que extablecerse
    - aguas
    - residuos solidos
    - plagas
    - desinfección

POE
procesos que se debe hacer diariamente por cada área

invima, ica (carnes/lacteos)

## Primeros requrimientos

Sistema de alarmas por cada proceso y check list de los procesos que se hayan realizado cada día.
Tomar fotos y almacenarlas como evidencia
Enviar mensualmente o trimestralmente un informe de cómo va ese proceso, para saber si ha mejorado o si se ha estado usando la app y cómo para obtener feedback constante.

La idea es tener una sistema (web o mobile (tablets o smarthphones)) donde se despliega el mapa del lugar con cada área. Cuando se hace clic en el area, entonces los procesos correspondientes así como la guia y/o notas a seguir en caso de olvidarlo.

Hay varios perfiles, el cliente que no puede editar los procesos, solo checkearlos y los administradores que pueden hacer cambios y contruir el plan de saneamiento.

Un ejemplo de plan de saneamiento está el archivo POE-LYD ArtesaPan 14 de julio.docx en el folder contenedor de este archivo.
